JSON (JavaScript Object Notation) — это легковесный формат для обмена данными, который легко читается и 
записывается человеком и парсится и генерируется машинами. JSON часто используется для передачи данных между 
клиентом и сервером. В AJAX-запросах данные обычно отправляются в формате JSON, используя `JSON.stringify()` 
для конвертации JavaScript-объекта в строку перед отправкой.
