1.AJAX (Asynchronous JavaScript and XML) — это техника разработки, позволяющая асинхронно взаимодействовать с 
сервером, обновляя данные на веб-странице без перезагрузки. Это достигается за счёт использования JavaScript 
для отправки запросов и получения данных с сервера, обычно в формате XML или JSON.
Что такое AJAX?
AJAX (Asynchronous JavaScript and XML) — это метод обновления веб-страниц асинхронно, без необходимости перезагрузки страницы.
Он позволяет обмениваться данными с сервером и обновлять части веб-страницы.
