15. Что такое абстрактный класс и для чего он нужен?
Абстрактный класс — это класс, который не может быть инстанцирован и предназначен только для наследования. 
Он обычно содержит один или несколько абстрактных методов, которые должны быть реализованы в дочерних классах. 
В JavaScript нет встроенной поддержки абстрактных классов, но их можно эмулировать с помощью выбрасывания ошибок.
