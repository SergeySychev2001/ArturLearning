4. Как в JavaScript реализуется наследование?
В JavaScript наследование реализуется через прототипы. В объекте может быть ссылка на другой объект (прототип), и 
свойства/методы родительского объекта будут доступны дочернему объекту. Есть также возможность использовать классы 
(с помощью ключевого слова `class`) для более понятного синтаксиса.
