В JavaScript конструкция `for...of` предоставляет удобный способ итерации по итерируемым объектам, таким как массивы, 
строки, массивоподобные объекты (например, `NodeList`), множества (`Set`), и сопоставимые объекты (`Map`).

Синтаксис использования `for...of` выглядит следующим образом:

javascript
for (const element of iterable) {
    // Ваш код, использующий element
}


Здесь `iterable` — это объект, который поддерживает итерацию, а `element` — переменная, в которую 
на каждой итерации будет записываться очередной элемент.

### Примеры использования `for...of`

1. Итерация по массиву
2. Итерация по строке
3. Итерация по множеству (Set)
4. Итерация по объекту Map

Основные моменты
-for...of не работает с обычными объектами (например, с `{}`), так как они не являются итерируемыми. 
Чтобы итерироваться по объектам, можно использовать `for...in`, чтобы получить ключи, или методы `Object.keys()`, 
Object.values()`, `Object.entries()` для получения массивов.
-for...of подходит для получения значений элементов. Если нужно получать индексы (например, во время работы с массивами), 
лучше использовать `for...in` или классическую конструкцию `for`.
