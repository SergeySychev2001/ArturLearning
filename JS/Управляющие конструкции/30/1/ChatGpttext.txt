`for...in` и `for...of` — это два разных цикла в JavaScript, которые используются для перебора элементов, но между ними есть важные отличия.

### for...in
- Используется для перебора **ключей** (или индексов) объектов и массивов.
- Итерации производятся по всем перечисляемым свойствам объекта.
- Синтаксис:
javascript
  for (let key in object) {
      console.log(key, object[key]);
  }
  


**Пример:**
javascript
const obj = { a: 1, b: 2, c: 3 };
for (let key in obj) {
    console.log(key, obj[key]); // Выведет ключи и значения: a 1, b 2, c 3
}


### for...of
- Используется для перебора **значений** итерабельных объектов, таких как массивы, строки, `Map`, `Set` и т.д.
- Итерации производятся по значениями элементов.
- Синтаксис:
javascript
  for (let value of iterable) {
      console.log(value);
  }
  


**Пример:**
javascript
const arr = [1, 2, 3];
for (let value of arr) {
    console.log(value); // Выведет значения: 1, 2, 3
}


### Основные отличия
1. **Тип перебираемого объекта**:
- `for...in` работает с объектами и получает ключи.
- `for...of` работает с итерабельными объектами и получает значения.

2. **Список перебираемых элементов**:
- `for...in` также может перебрать унаследованные свойства объекта.
- `for...of` перебирает только собственные элементы итерабельного объекта.

3. **Использование**:
- Используйте `for...in` для работы с объектами.
- Используйте `for...of` для работы с массивами и другими итерабельными структурами.
