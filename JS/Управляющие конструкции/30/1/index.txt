- for...in предназначен для перебора свойств объекта (ключей), в том числе унаследованных. С 
его помощью можно итерироваться по массивам, но рекомендуется для объектов.
- for...of используется для перебора итерируемых объектов, таких как массивы, строки, карты, 
множества и т.д. Этот цикл перебирает значения элементов.


Основная разница между этими операторами заключается в том, что for...in проходит по всем перечисляемым свойствам объекта 
(включая свойства, унаследованные от прототипа), 
тогда как for...of проходит только по значениям элементов объекта, которые доступны через итератор.
