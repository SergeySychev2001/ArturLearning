21. Как работают генераторы в ES6 и для чего они нужны?
Генераторы в ES6 позволяют создавать итераторы с помощью функции, которая 
возвращает на каждом вызове значение с помощью ключевого слова `yield`. Генераторы объявляются с использованием `function*`.
Генераторы полезны для работы с потоками данных, асинхронным программированием и управлением состоянием.
