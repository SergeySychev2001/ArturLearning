Блочная область видимости (block scope) в JavaScript — это концепция, при которой переменные определяются в 
пределах блока кода, заключенного в фигурные скобки {}. В ES6 (EcmaScript 2015) были введены две новые ключевые буквы — let и const, 
которые позволяют создавать переменные с блочной областью видимости.

Как это работает
let: Переменная, объявленная с помощью let, существует только внутри блока, где она была объявлена. 
Это означает, что она недоступна вне этого блока.

const: Переменная, объявленная с помощью const, также имеет блочную область видимости. Однако, в отличие от let, const 
используется для объявления констант, т.е. значений, которые не могут быть переназначены
