`async function*` в JavaScript — это синтаксис для создания асинхронного генератора. Асинхронные генераторы позволяют вам 
возвращать значения асинхронно с помощью `await` и генерировать последовательности значений с помощью `yield`. Эти функции полезны 
в сценариях, когда нужно обрабатывать данные по мере их поступления и ожидать завершения асинхронных операций, прежде чем продолжить.

Основные характеристики `async function*`
1. Асинхронный контекст: Внутри асинхронного генератора можно использовать оператор `await`, что позволяет ожидать завершения асинхронных операций.
2. Генерация значений: При помощи `yield` можно генерировать значения. С помощью `for await...of` можно итерировать по последовательности значений.
3. Интеграция с асинхронными итераторами: Асинхронные генераторы позволяют создавать собственные асинхронные итераторы.
