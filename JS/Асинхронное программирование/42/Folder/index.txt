Асинхронные итераторы в JavaScript позволяют работать с асинхронными последовательностями данных, что особенно полезно 
при работе с потоками данных, такими как получение данных от API или чтение файлов. Асинхронные итераторы позволяют использовать 
`for await...of` для итерации по данным, поступающим асинхронно.
