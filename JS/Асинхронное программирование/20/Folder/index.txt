В JavaScript вы можете отменить таймеры, созданные с помощью setTimeout и setInterval, используя функции 
clearTimeout и clearInterval соответственно. Эти функции принимают идентификатор таймера, который возвращается 
при вызове setTimeou` или setInterval.

Как работает clearTimeout
Функция clearTimeout используется для отмены выполнения функции, которая должна была быть вызвана через setTimeout.

Синтаксис:
javascript
clearTimeout(timeoutId);

- timeoutId — идентификатор таймера, возвращаемый setTimeout


Как работает clearInterval

Функция clearInterval используется для остановки повторного выполнения функции, которая была запланирована с помощью setInterval.

Синтаксис:
javascript
clearInterval(intervalId);
- intervalId — идентификатор интервала, возвращаемый setInterval


Важные замечания:
- `clearTimeout` и `clearInterval` не возвращают никаких значений, поэтому их нельзя использовать для проверки, было 
ли действие успешно выполнено.
- Если идентификатор таймера, переданный в `clearTimeout` или `clearInterval`, не существует (например, если он уже был очищен), 
функция просто ничего не сделает. Это не вызовет ошибок.
- Важно помнить о правильном использовании идентификаторов таймеров, чтобы избежать их очистки до того, как они будут фактически
 запланированы.
