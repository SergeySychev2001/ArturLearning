XMLHttpRequest` — это встроенный объект в JavaScript, который позволяет выполнять асинхронные HTTP-запросы 
к серверу. Он был одним из первых способов осуществления AJAX (Asynchronous JavaScript and XML) вызовов, что 
позволяет загружать данные без перезагрузки всей страницы.

Хотя в современном мире разработчики чаще используют API Fetch для выполнения асинхронных запросов, `XMLHttpRequest` все 
еще поддерживается и может быть полезен для понимания основ работы с HTTP-запросами.

Синтаксис XMLHttpRequest
Чтобы использовать XMLHttpRequest, следуйте этим шагам:
1. Создайте новый объект `XMLHttpRequest`.
2. Настройте запрос с помощью метода `open()`.
3. Определите, что делать при получении ответа, используя свойство `onreadystatechange`.
4. Отправьте запрос с помощью метода `send()`.



Заключение

`XMLHttpRequest` является мощным инструментом для выполнения асинхронных HTTP-запросов. Однако в современных 
приложениях часто предпочтительнее использовать `fetch API`, который предоставляет более современный и удобный синтаксис 
для работы с сетевыми запросами. Тем не менее, изучение `XMLHttpRequest` полезно для понимания основ клиент-серверной архитектуры в JavaScript
