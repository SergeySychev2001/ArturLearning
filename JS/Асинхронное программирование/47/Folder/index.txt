Асинхронные генераторы в JavaScript позволяют обрабатывать потоки данных асинхронно, что делает их полезными для работы с 
данными, которые могут поступать с задержками, например, из сетевого источника или из другой асинхронной библиотеки.

Основные концепции:
1. Асинхронные итераторы: Это объект, который реализует метод `Symbol.asyncIterator`. Он возвращает объект, который имеет 
метод `next()`, возвращающий промис, резолвящийся в объект с двумя свойствами: `value` и `done`.
2. Асинхронные генераторы: Это функция, объявленная с ключевым словом `async function*`. Она позволяет использовать 
`yield` для генерации значений асинхронно.
