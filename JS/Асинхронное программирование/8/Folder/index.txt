Методы then, catch и finally являются стандартными методами, которые используются для обработки результатов работы Promise. 
Давайте рассмотрим каждый из них подробнее.

1. then()
Метод then() используется для обработки успешного завершения Promise (то есть, когда он находится в состоянии "исполнен"). 
Он принимает два аргумента:
- Первый аргумент (обязательный) — функция, которая будет вызвана при успешном завершении, с результатом `Promise` в качестве аргумента.
- Второй аргумент (необязательный) — функция для обработки ошибки, которая будет вызвана, если `Promise` был отклонен.


2. catch()
Метод catch() используется для обработки ошибок, возникающих в любом из предыдущих then(). Он принимает 
один аргумент — функцию, которая будет вызвана, если `Promise` был отклонен. Это позволяет обрабатывать ошибки в цепочке `Promise`.

3. finally()
Метод finally() позволяет выполнить код после завершения Promise, независимо от его состояния 
(исполнен или отклонен). Это полезно для выполнения каких-либо завершающих действий, например, скрытия загрузчика.
