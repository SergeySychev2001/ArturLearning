Отправка данных на сервер с помощью Fetch API производится, как правило, с использованием метода `POST`, но 
также могут использоваться и другие методы, такие как `PUT` и `PATCH`. Вот несколько примеров того, как это сделать.

Заключение
Fetch API предлагает простой и мощный способ отправлять данные на сервер. Будьте внимательны при работе с заголовками, форматом 
данных и методами HTTP, чтобы ваши запросы были выполнены корректно.
