Область видимости переменной в JavaScript — это контекст, в котором переменная объявлена и доступна для использования. 
В JavaScript есть два основных типа области видимости:

Глобальная область видимости: Переменные, объявленные вне функций, доступны отовсюду в коде.

Локальная область видимости: Переменные, объявленные внутри функции, доступны только внутри этой функции и её подфункциях.

С появлением ES6 добавились ещё два типа:

Блочная область видимости: Переменные, объявленные с помощью let и const, ограничены блоком, в котором они были объявлены (
    например, циклом или условным оператором).

Область видимости модуля: Переменные, объявленные в модуле, доступны только внутри этого модуля, если только они явно не экспортируются.
