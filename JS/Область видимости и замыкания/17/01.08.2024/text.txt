В JavaScript ключевое слово `this` связывается с контекстом выполнения функции, 
который может изменяться в зависимости от того, как функция была вызвана. Вот основные 
способы, как this определяется в различных областях видимости:
1.Глобальный контекст
В глобальной области видимости (т.е. вне всех функций) this ссылается на глобальный объект. В браузерах это объект window.
2.Контекст функции
В обычной функции (не стрелочной) при вызове this будет ссылаться на объект, который вызвал функцию. 
Если функция была вызвана как метод объекта, то this будет ссылаться на этот объект.
3.Стрелочные функции
Стрелочные функции не имеют собственного this. Они захватывают this из своего лексического окружения 
(области видимости, где они были определены).
4.Методы call, apply и bind:
Эти методы позволяют явно задавать значение this в любой функции.
5.Объекты класса
В классах this будет указывать на экземпляр класса.
6.Условия и обработка событий
В обработчиках событий this ссылается на элемент, который вызвал событие, если не используется стрелочная 
функция или не переопределено значение с помощью bind.
